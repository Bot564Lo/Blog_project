Introducción a HTML

La creación de sitios web modernos y dinámicos comienza con una base sólida: HTML. HTML, o Lenguaje de Marcado de Hipertexto, es el lenguaje estándar para diseñar y estructurar el contenido en la web. Si estás comenzando tu viaje en el desarrollo web, entender HTML es el primer paso esencial.

¿Qué es HTML?

HTML es un lenguaje de marcado utilizado para estructurar y presentar contenido en la web. A diferencia de los lenguajes de programación como JavaScript o Python, HTML no tiene lógica ni funcionalidad por sí mismo. En cambio, define la estructura y el diseño de una página web utilizando una serie de elementos y atributos.

La Estructura Básica de un Documento HTML

Un documento HTML típico comienza con la declaración del tipo de documento (<!DOCTYPE html>) y está compuesto por varios elementos anidados. Aquí hay un ejemplo de la estructura básica: 

 
                    <!DOCTYPE html>
                    <html lang="es">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Mi Primera Página Web</title>
                    </head>
                    <body>
                        <h1>¡Hola, mundo!</h1>
                        <p>Bienvenido a mi primera página web.</p>
                    </body>
                    </html>

Desglose de la Estructura

  • <!DOCTYPE html>: Esta declaración define el tipo de documento y asegura que el navegador interprete el contenido como HTML5.
  • <html lang="es">: El elemento <html> es el contenedor raíz de todo el documento. El atributo lang especifica el idioma del contenido.
  • <head>: El elemento <head> contiene metadatos sobre el documento, como el conjunto de caracteres (<meta charset="UTF-8">), la configuración de la vista (<meta name="viewport" content="width=device-width, initial-scale=1.0">), y el título de la página (<title>).
  • <body>: El elemento <body> contiene todo el contenido visible de la página web, como encabezados (<h1>), párrafos (<p>), imágenes (<img>), enlaces (<a>), y más.

Elementos Comunes de HTML

A medida que profundices en HTML, te encontrarás con muchos elementos y atributos que te permitirán crear estructuras complejas. Aquí hay algunos de los elementos más comunes:

  • Encabezados: <h1> a <h6> para títulos y subtítulos.
  • Párrafos: <p> para bloques de texto.
  • Enlaces: <a href="url"> para crear hipervínculos.
  • Imágenes: <img src="ruta" alt="descripción"> para insertar imágenes.
  • Listas: <ul> (no ordenada) y <ol> (ordenada) con <li> para elementos de lista.
  • Divisiones y secciones: <div> y <section> para agrupar contenido.

CSS: Dando estilo a tu web

Si HTML es el esqueleto de una página web, CSS (Hojas de Estilo en Cascada) es la piel que le da vida y color. CSS se utiliza para controlar la presentación de un documento escrito en HTML o XML, permitiendo separar el contenido de la presentación y proporcionando un control total sobre el diseño visual de una página web.

¿Qué es CSS?

CSS es un lenguaje de diseño que describe la presentación de un documento estructurado en HTML o XML. Permite aplicar estilos como colores, fuentes, y espaciado a los elementos de la página, y también manejar el diseño y la disposición de los mismos.

La Estructura Básica de una Regla CSS

Una regla CSS está compuesta por un selector y una declaración. El selector indica qué elementos HTML serán estilizados, y la declaración define cómo serán estilizados esos elementos. Aquí hay un ejemplo básico:


                    h1 {
                        color: blue;
                        font-size: 24px;
                    }

Desglose de la Regla CSS

  • h1: El selector que indica que esta regla CSS se aplicará a todos los elementos <h1> en el documento.
  • color: blue;: Una declaración que establece el color del texto de los elementos <h1> en azul.
  • font-size: 24px;: Otra declaración que establece el tamaño de la fuente de los elementos <h1> en 24 píxeles.

Metodologías de Inclusión de CSS

Existen varias formas de incluir CSS en un documento HTML:

CSS en Línea

Se utiliza el atributo style directamente en el elemento HTML. Este método no se recomienda para estilos grandes debido a la falta de separación entre el contenido y el diseño.

<h1 style="color: blue; font-size: 24px;">Título en Azul</h1>

CSS Interno

Se incluye dentro de una etiqueta <style> en el <head> del documento HTML. Es útil para estilos específicos de una sola página.


                    <head>
                        <style>
                            h1 {
                                color: blue;
                                font-size: 24px;
                            }
                        </style>
                    </head>

CSS Externo

Se escribe en un archivo separado con extensión .css y se vincula al documento HTML mediante la etiqueta <link>. Este es el método más recomendado para proyectos grandes y para mantener un código limpio y organizado.


                    <head>
                        <link rel="stylesheet" href="styles.css">
                    </head>

Propiedades Comunes de CSS

A continuación, algunas de las propiedades CSS más comunes:

  • Color: color, background-color
  • Texto: font-family, font-size, text-align, text-decoration
  • Espaciado: margin, padding
  • Diseño: display, position, flex, grid
  • Otros: border, width, height

JavaScript: Añadiendo interactividad

Si HTML es el esqueleto y CSS la piel, JavaScript es el cerebro que añade interactividad a tu página web. JavaScript es un lenguaje de programación que permite implementar características complejas en las páginas web, como formularios interactivos, animaciones, y comunicación en tiempo real con servidores.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que se ejecuta en el navegador del usuario. Permite manipular el Document Object Model (DOM), respondiendo a eventos y creando experiencias de usuario dinámicas y atractivas.

La Estructura Básica de un Script JavaScript

Un script JavaScript puede incluirse directamente en el archivo HTML o en un archivo separado. Aquí hay un ejemplo básico de un script que muestra un mensaje en la consola del navegador:


                    <script>
                        console.log("¡Hola, mundo!");
                    </script>

Desglose del Script JavaScript

  • <script>: La etiqueta que contiene el código JavaScript. Puede colocarse en el <head> o al final del <body>.
  • console.log("¡Hola, mundo!");: Un comando que imprime un mensaje en la consola del navegador, útil para depuración.

Metodologías de Inclusión de JavaScript

Existen varias formas de incluir JavaScript en un documento HTML:

JavaScript en Línea

Se utiliza el atributo onclick directamente en un elemento HTML. Este método es útil para pequeños scripts, pero no se recomienda para aplicaciones más complejas.

JavaScript Interno

Se incluye dentro de una etiqueta <script> en el documento HTML. Es útil para scripts específicos de una sola página.


                <head>
                    <script>
                        function greet() {
                            alert('¡Hola, mundo!');
                        }
                    </script>
                </head>
                <body>
                    <button onclick="greet()">Haz clic</button>
                </body>

JavaScript Externo

Se escribe en un archivo separado con extensión .js y se vincula al documento HTML mediante la etiqueta <script>. Este es el método más recomendado para mantener un código limpio y organizado.


                    <head>
                        <script src="script.js"></script>
                    </head>

Funciones y Eventos en JavaScript

A continuación, algunos conceptos clave de JavaScript:

Funciones

Una función es un bloque de código diseñado para realizar una tarea específica. Se define utilizando la palabra clave function.


                function greet() {
                    alert('¡Hola, mundo!');
                }

Eventos

Los eventos son acciones que ocurren en la página web, como un clic del usuario, y pueden ser manejados con JavaScript para crear interactividad.


                document.getElementById("myButton").addEventListener("click", function() {
                    alert('¡Hola, mundo!');
                });

Frameworks: Herramientas Poderosas para Desarrolladores Web

Los frameworks son conjuntos de herramientas y bibliotecas que proporcionan estructuras y funcionalidades predefinidas para facilitar el desarrollo web. Estos frameworks permiten a los desarrolladores construir aplicaciones web más rápidamente al proporcionar soluciones preconfiguradas para tareas comunes como el enrutamiento, la gestión del estado, y la interacción con servidores.

¿Qué son los Frameworks?

Un framework web es una estructura de software diseñada para facilitar el desarrollo y la implementación de aplicaciones web. Proporcionan una arquitectura organizada y predefinida que guía el flujo de trabajo del desarrollo, promoviendo buenas prácticas y patrones de diseño.

Beneficios de Utilizar Frameworks

  • Productividad Mejorada: Los frameworks proporcionan componentes reutilizables y funciones integradas que aceleran el desarrollo.
  • Consistencia y Mantenibilidad: Promueven la estructura organizada del código y facilitan la colaboración en equipos.
  • Seguridad Mejorada: Incorporan prácticas de seguridad recomendadas y actualizaciones frecuentes para proteger las aplicaciones.

Tipos de Frameworks

  • Frontend: Se centran en la interfaz de usuario y la interacción del cliente en el navegador. Ejemplos populares incluyen React, Angular y Vue.js.
  • Backend: Se enfocan en la lógica de negocio y la gestión de datos en el servidor. Ejemplos notables son Express.js para Node.js y Django para Python.
